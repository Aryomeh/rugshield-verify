GAME CONTROLS

Pause, resume, or reset the live game

⏸ Pause / Resume Game

Sets game_paused = true in Supabase. Your app.js already has isPaused — hook it to this flag on init.

🔄 Reset Leaderboard

Deletes all rows from the scores table. The leaderboard view resets automatically. Cannot be undone.

🔓 Re-lock Onboarding

Sets is_verified = false for every user, forcing all players to redo onboarding tasks.
